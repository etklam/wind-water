# 道家玄學八卦盤 UI 重寫設計（全頁）

## 背景與目標
- 專案目前可完成納音五行計算，但整體視覺偏通用表單。
- 本次重寫目標是以「中國傳統道家玄學」為主題，並以「五行 + 八卦」作為視覺中心。
- 範圍為整體 UI（首頁全頁），非單一區塊微調。

## 設計方向定案
- 風格：道家玄學（太極八卦盤、深色、神秘）
- 主方案：八卦盤主視覺（非卷軸式、非科技儀表板）

## 整體資訊架構
### 1. 三層視覺結構
1. 玄色背景層：夜空玄色漸層、低透明符紋與雲氣。
2. 主內容層：輸入與結果主體。
3. 符紋裝飾層：不干擾操作的輕量裝飾元素。

### 2. 版面配置
- 桌機：雙欄
1. 左欄：輸入表單（出生日期/時間/時區/模式/語言）
2. 右欄：太極八卦盤 + 五行結果主舞台
- 手機：單欄堆疊（表單在上，八卦盤在下）

### 3. 結果呈現
- 八卦盤中心固定太極圖，外圈固定八卦符號。
- 五行統計映射到五方位節點：
1. 東：木
2. 南：火
3. 中：土
4. 西：金
5. 北：水
- 四柱資料放到八卦盤下方卡片列，避免視覺競爭。

## 視覺語言規格
### 1. 色彩系統
- 主背景：`#0B0F14` -> `#111A22`
- 主金色：`#C8A96B`（八卦線條、標題）
- 靈青色：`#3AA6A0`（互動與高亮）
- 五行色：
1. 木 `#4F8F5A`
2. 火 `#C75B39`
3. 土 `#A4875A`
4. 金 `#B7B9BF`
5. 水 `#3C6E8F`

### 2. 字體系統
- 標題：`Noto Serif TC`
- 內文：`Noto Sans TC`
- 數值：等寬或半等寬字體

### 3. 材質與細節
- 背景加入淡顆粒與符紋圖層，避免平面化。
- 卡片使用深色半透明、細金邊、內陰影。
- 互動按鈕使用短時青金掃光，避免過度動畫。

## 互動與動效規則
### 1. 主要互動
- 點擊「開始計算」後，八卦盤出現由內向外微光擴散（約 600ms）。
- 五行節點依計算值改變亮度與脈衝幅度。
- 最高元素節點額外加金青光環。
- 五行節點支援 hover/tap 資訊提示（元素名 + 數值）。

### 2. 動效節奏
- 首屏：標題與表單先進場，八卦盤延遲 120ms 淡入。
- 結果刷新：使用 `opacity + translateY` 180-240ms。
- 緩動函數：統一 ease-out，維持儀式感。

### 3. 響應式與可用性
- 手機版八卦盤等比縮放，五行資訊增加盤下橫向標籤避免小字。
- 所有互動元素保留可見 focus 樣式，支援鍵盤操作。
- 勾選未知時辰時，時柱區塊保留但以「未提供」弱化呈現。

## 非目標（YAGNI）
- 本次不加入占卜解讀文案生成。
- 本次不做八卦節點拖曳或自定義布局。
- 本次不引入外部圖形引擎（如 canvas/svglib 依賴）。

## 驗收標準
1. 頁面整體視覺明顯為道家玄學風格。
2. 五行與八卦在首屏可見且為核心焦點。
3. 桌機與手機皆可正常使用，輸入與結果流程不退化。
4. 現有計算邏輯與 i18n 行為維持正確。
